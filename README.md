# 📈 Stock Price Prediction using Machine Learning

This project focuses on predicting the next day's closing price of a stock using historical market data. The data is fetched using the yfinance API and processed using Python-based data science tools.

## 🚀 Features
- Real-time stock data fetching using Yahoo Finance API
- Feature engineering (Price Change, High-Low Difference)
- Machine Learning models:
  - Linear Regression
  - Random Forest Regressor
- Time-series aware train-test splitting
- Visualization of actual vs predicted prices
- Next-day stock price prediction
- Model saving using joblib

## 🧠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- yfinance

## 📊 Output
- Comparison graph of actual vs predicted stock prices
- Predicted next-day closing price

## 📌 Use Case
This project demonstrates practical application of machine learning in financial forecasting and time-series data analysis.
